Steps to run the code for


1.Extract all the files into a separate folder.
2.Note the location where the gallery and probe images in *.pgm format saved. 
3.Replace the locations in lines 6 and 138 for gallery and probe images in “main.m”.
4.Replace the location to where you would like to save the eigenfaces in line 4 “disp_img.m”
5. Now run the program by giving the following commands

main

6.This will run all the tests and save the results with the images in a demo.mat workspace which can be checked later.

7. The images folder contains the plots produced for further reference. These plots are also available towards the end of the report.
